# Security Policy

## Supported Versions

This project is provided for research and educational experimentation. Security updates are handled on a best-effort basis.

## Reporting a Vulnerability

Please do not open a public issue for security vulnerabilities.

Report vulnerabilities by email:

- 70169515@student.uol.edu.pk

Please include:

- Description of the issue
- Steps to reproduce
- Potential impact
- Any suggested fix

You will receive an acknowledgment as soon as possible.
